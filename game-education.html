By clicking on the collapse component, I expect it to open, but it doesn't, so I have to click on the 'show' element. Also, I would move the arrow from the show elements to the same level as the title, so that when opening it, the height of the collapse cap would not be cut. When setting background-image , it is necessary to set a contrasting background color so that when loading an image, it understands that there should be something there. Screenshots on mobile devices are not informative. It is desirable to make a pop-up window with readable information when clicking on a thumbnail.
