Game Updates

In an earlier game update, the patch size is specified, which improves the user experience for players with limited Internet access speed and/or a limited amount of traffic. Immediately after the block about limiting server operation due to the update, the new version has a list of changes to briefly review all the changes. I would do a smooth scrolling to the desired block by clicking on a link in the navigation.

The new version has collapse components that visually unload the interface and, if necessary, allow you to get acquainted in detail with the features in the presence of inters.

In conclusion, I want to say that I like the new version of the article about the game update more, it is more user-friendly, less loaded with colorful screenshots for the entire width of the article, it allows you to quickly get acquainted with the main changes, and, if necessary, learn more with the help of collapses.
